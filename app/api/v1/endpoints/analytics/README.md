# Analytics Module

This module consolidates all reporting and business intelligence features.

## Components
- **General Reports**: Standard financial and operational reports (`general.py`).
- **Custom Reports**: User-defined dynamic reports (`custom.py`).
- **YoY Reports**: Year-over-Year comparison analysis (`yoy.py`).
- **BI / Metabase**: Metabase integration for dashboard embedding (`bi.py`).

## Usage
Endpoints: `/reports`, `/custom-reports`, `/yoy-reports`, `/bi`.
